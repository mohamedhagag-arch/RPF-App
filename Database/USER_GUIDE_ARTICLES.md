<h1>Creating Your First KPI</h1>

<p>Key Performance Indicators (KPIs) are essential metrics that help you track the progress and success of your projects. This guide will walk you through creating your first KPI.</p>

<h2>What is a KPI?</h2>
<p>A KPI is a measurable value that demonstrates how effectively a project is achieving key objectives. KPIs help you:</p>
<ul>
  <li>Track project progress</li>
  <li>Identify areas that need attention</li>
  <li>Make data-driven decisions</li>
  <li>Measure success against goals</li>
</ul>

<h2>Creating a KPI - Step by Step</h2>

<h3>Step 1: Navigate to KPI Management</h3>
<p>From the main menu, select <strong>KPI</strong> → <strong>Add KPI</strong> or use the Smart KPI Form.</p>

<h3>Step 2: Select Project</h3>
<p>Choose the project for which you want to create a KPI. You can search for projects by name or code.</p>

<h3>Step 3: Enter Activity Details</h3>
<p>Fill in the following information:</p>
<ul>
  <li><strong>Activity Name:</strong> A descriptive name for the activity</li>
  <li><strong>Input Type:</strong> Select the type of input (Quantity, Rate, etc.)</li>
  <li><strong>Quantity:</strong> Enter the quantity value</li>
  <li><strong>Unit:</strong> Select the appropriate unit of measurement</li>
</ul>

<h3>Step 4: Set Dates</h3>
<p>Specify the planned and actual dates for the activity. This helps track timeline adherence.</p>

<h3>Step 5: Add Additional Information</h3>
<p>You can optionally add:</p>
<ul>
  <li>Activity timing</li>
  <li>Activity division</li>
  <li>Activity scope</li>
  <li>Notes or comments</li>
</ul>

<h3>Step 6: Save</h3>
<p>Click <strong>Save</strong> to create your KPI. The system will automatically calculate values based on your inputs.</p>

<h2>Best Practices</h2>
<ul>
  <li>Use clear, descriptive names for activities</li>
  <li>Ensure quantities are accurate</li>
  <li>Update KPIs regularly</li>
  <li>Review and approve KPIs in a timely manner</li>
</ul>

<h2>Next Steps</h2>
<p>After creating a KPI, you can:</p>
<ul>
  <li>View it in the KPI Tracking page</li>
  <li>Edit it if changes are needed</li>
  <li>Submit it for approval</li>
  <li>Track its progress over time</li>
</ul>

<h1>Understanding Bill of Quantities (BOQ)</h1>

<p>The Bill of Quantities (BOQ) is a document used in construction and project management that provides measured quantities of the items of work identified in the drawings and specifications.</p>

<h2>What is BOQ?</h2>
<p>BOQ is a comprehensive list of:</p>
<ul>
  <li>Materials required for a project</li>
  <li>Labor needed</li>
  <li>Equipment and tools</li>
  <li>Associated costs and rates</li>
</ul>

<h2>Why is BOQ Important?</h2>
<p>BOQ helps you:</p>
<ul>
  <li><strong>Estimate Costs:</strong> Accurately estimate project costs</li>
  <li><strong>Plan Resources:</strong> Plan materials and labor requirements</li>
  <li><strong>Track Progress:</strong> Monitor actual vs. planned quantities</li>
  <li><strong>Control Budget:</strong> Keep project costs under control</li>
  <li><strong>Generate Reports:</strong> Create detailed cost reports</li>
</ul>

<h2>BOQ Structure</h2>
<p>A typical BOQ includes:</p>

<h3>1. Activity Information</h3>
<ul>
  <li>Activity name and description</li>
  <li>Activity code</li>
  <li>Category and classification</li>
</ul>

<h3>2. Quantity Details</h3>
<ul>
  <li>Planned quantity</li>
  <li>Actual quantity</li>
  <li>Unit of measurement</li>
  <li>Variance</li>
</ul>

<h3>3. Rate Information</h3>
<ul>
  <li>Unit rate</li>
  <li>Total cost</li>
  <li>Material cost</li>
  <li>Labor cost</li>
  <li>Equipment cost</li>
</ul>

<h3>4. Status and Progress</h3>
<ul>
  <li>Completion percentage</li>
  <li>Status (Planned, In Progress, Completed)</li>
  <li>Timeline information</li>
</ul>

<h2>Working with BOQ in the Application</h2>

<h3>Viewing BOQ</h3>
<p>Navigate to <strong>BOQ</strong> from the main menu to view all BOQ activities. You can:</p>
<ul>
  <li>Filter by project</li>
  <li>Search for specific activities</li>
  <li>Sort by various criteria</li>
  <li>Export data</li>
</ul>

<h3>Adding BOQ Activities</h3>
<p>To add a new BOQ activity:</p>
<ol>
  <li>Click <strong>Add Activity</strong></li>
  <li>Select the project</li>
  <li>Enter activity details</li>
  <li>Set quantities and rates</li>
  <li>Save the activity</li>
</ol>

<h3>Editing BOQ</h3>
<p>You can edit BOQ activities to:</p>
<ul>
  <li>Update quantities</li>
  <li>Modify rates</li>
  <li>Change status</li>
  <li>Add notes</li>
</ul>

<h2>Best Practices</h2>
<ul>
  <li>Keep BOQ data up to date</li>
  <li>Regularly review variances</li>
  <li>Document changes and reasons</li>
  <li>Link BOQ to project KPIs</li>
  <li>Use consistent naming conventions</li>
